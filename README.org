El proyecto se compila con 'mvn clean install', y se corre con 'mvn tomcat7:run'
y poniendo en un navegador 'http://localhost:8080/infomex/'.

La base de datos se crea con el nombre 'infomex', y una vez dentro de ella se debe
ejecutar el archivo 'baseDeDatos.sql'.
